Extract from Wikipedia page on life expectancy

Part of a Wikipedia page, List of countries by life expectancy, has been reproduced here to demonstrate how pigshell can extract, filter and display table data from a web page.

Since Wikipedia does not support CORS, so pigshell's Javascript cannot retrieve Wikipedia pages due to the same origin policy. The solution is to run the psty proxy server on your desktop, so pigshell can access any resource exposed via HTTP.

However, we want examples to be accessible without requiring psty, so we have mirrored the page under the pigshell.com domain.
